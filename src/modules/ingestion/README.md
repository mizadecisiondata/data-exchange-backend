# Ingestion Module

Responsabilidad futura:

- Ingerir bloques de informacion desde CSV, Excel, API y SFTP si aplica.
- Validar calidad minima del 95%.
- Generar reporte de errores.
- Descartar duplicados sin contarlos como error ni generar Decision Credits.
- Preparar datos para almacenamiento analitico y consulta.

No usar el concepto de "bloque de cartera" como nombre principal del producto.
